MySQL 常用函数

TenIncense · 2022-7-3 · 次阅读

一、concat

1、concat(str1,str2)

concat(str1,str2) 返回为连接参数后产生的字符串，如果参数中有 null，则返回结果为 null

例：

  • 参数中没有 null 

    SELECT concat('My','S','QL');
> 'MySQL'

  • 参数中有 null 

    SELECT concat('My',NULL,'QL');
> NULL

2、concat_ws(separator,str1,str2,…)

concat_ws(separator,str1,str2,...) 类似于 concat，第一个参数是分隔符，分隔符会放在要连接的两个字符串之间，分隔符可以是字符串也可以是任意参数。如果分隔符为 NULL，则整个结果为 NULL。除了分隔符外的任何参数为 NULL，NULL 值会被忽略；

例：

  • 分隔符使用逗号连接

    SELECT concat_ws(',','First name','Second name','Last Name');
> First name,Second name,Last Name

  • 分隔符参数使用 null 连接

    SELECT concat_ws(null,'First name','Second name','Last Name');
> NULL

  • 分隔符参数不为 null，但是其它参数为 null

    SELECT concat_ws(',','First name',NULL,'Last Name');
> First name,Last name

二、space(N)

返回空格组成的字符串。

例

  • 想要 6 个空格位置

    SELECT SPACE(6);
> '      '

三、strcmp(expr1,expr2)

如果 expr1 大于 expr2 返回 1，也就是 true。 如果 expr1 小于 expr2 返回 -1也就是 false。如果 expr1 等于 expr2 返回0；

例

  • expr1 小于 expr2

    SELECT STRCMP('text','text2');
> -1

  • expr1 大于 expr2

    SELECT STRCMP('text2','text');
> 1

  • *expr1 等于 expr2 *

    SELECT STRCMP('text','text');
> 0

四、length(str)

length(str) : 返回值为字符串 str 的字节长度

例：

  • 想知道一个字符串的长度是多少

    SELECT LENGTH('text');
> 4

五、left(str,len)

left(str,len) : 从左边开始截取指定长度的字符串

例：

  • 想从字符串中的左边截取 5个长度的字符串

    SELECT LEFT('foobarbar',5);
> fooba

六、right(str,len)

right(str,len) : 从右边开始截取指定长度的字符串

例：

  • 想从右边截取4 个长度的字符串

    SELECT RIGHT('foobarbar',4);
> rbar

七、upper(str)

upper(str) : 返回大写字母

例：

  • 给一个字符串升为大写

    SELECT UPPER('Hei');
> HEI

八、lower(str)

lower(str) : 返回小写字母

例：

  • 将一个字符串转为小写 

    SELECT LOWER('HELLO');
> hello

九、replace(str,str1,str2)

replace(str,str1,str2) : 将字符串str2替换掉 str 中的 str1

例：

  • 将每一个 w都替换成 a

    SELECT REPLACE('www.mysql.com','w','a');
>  aaa.mysql.com

十、reverse(str)

reverse(str) : 反转字符串，返回str字符串的顺序相反

例：

  • 反转一个字符串的顺序

    SELECT REVERSE('abc');
> cba

十一、row_count()

row_count() : 返回上一句sql增/删/改影响的记录数

例：

  • 先插入三条数据

    INSERT INTO t_b VALUES(1),(2),(3);
> row 3(影响行数)

  • 获取上一个插入语句影响了几条记录

    SELECT ROW_COUNT();
> 3

注意：如果上一条SQL 是查询语句（SELECT/SHOW 等）则返回-1；

如果上一条SQL 是删除表（DROP）则返回 0；

如果上一条SQL 是建表语句，且创建的表是空表则返回 0；

十二、found_rows()

found_rows() : 获取上一句SELECT语句（SHOW语句等）查询结果集的记录数

例：

  • 先查一次表中的记录

    SELECT * FROM teacher;
> 显示7行数据

  • 想知道上一次的查询记录数是多少

    SELECT FOUND_ROWS();
> 7

注意一下，这个FOUND_ROWS() 一般是搭配 SQL_CALC_FOUND_ROWS 用的

例：

# 如现在表中有3000万条数据
# 执行SQL_CALC_FOUND_ROWS的函数查询，查出3000 万条数据
SELECT SQL_CALC_FOUND_ROWS * FROM t_b;
> 得到3000万条数据显示

# 执行一次FOUND_ROWS()
SELECT FOUND_ROWS();
> 30000000 -- (得到总记录数)
  • SQL_CALC_FOUND_ROWS ： 意思是将 sql所处理的行数记录下来
  • FOUND_ROWS()：取到这个记录下来的数

注意：

1、这两句 SQL一定要放在一个 session 中，不然结果数据是错误的；

2、如果你要使用两句SQL 组合就不能用可视化软件 Navicat、SQLyon之类的执行这两句SQL 。

十三、if(expr1,expr2,expr3)

if(expr1,expr2,expr3) ：如果 expr1 是 true，则返回 expr2；如果 expr1 是 false 则返回 expr3

例：

  • 表达式 expr1 为 false 的时候

    SELECT IF(1>2,2,3);
> 3

  • 表达式 expr1 为 true 的时候

    SELECT IF(1<2,'yes','no');
> yes

十四、ifnull(expr1,expr2)

ifnull(expr1,expr2) : 当 expr1 不为 null，则返回 expr1，expr1为 null 则返回 expr2

例：

  • expr1 不为 NULL 的情况

    SELECT IFNULL(1,0);
> 1

  • expr1 为 NULL 的情况

    SELECT IFNULL(NULL,10);
> 10

十五、to_days(date)

to_days(date) : 给定一个日期date，返回一个天数（从年份 0 开始的天数）

例：

  • 从 0 开始到 ‘1997-01-02’ 距离是多少天

    SELECT TO_DAYS('1997-01-02');
> 729391

十六、datediff(expr1,expr2)

datediff(expr1,expr2) : 返回expr1时间和 expr2时间之间的间隔天数。

注意：注意参数顺序

例：

  • 两个参数中的最大的日期在第一位情况下查看间隔天数

    SELECT DATEDIFF('1997-12-31 23:59:59','1997-12-31');
> 1

  • 两个参数中最大的日期在第二位情况下查看间隔天数

    SELECT DATEDIFF('1997-11-30 23:59:59','1997-12-31');
> -31

十七、last_day(date)

last_day(date) : 返回该月最后一天对应的值。若参数 date无效，则返回NULL

例：

  • 获取2004-01-01 01:01:01的最后一天

    SELECT LAST_DAY('2004-01-01 01:01:01');
> 2004-01-31

  • DATE参数为非法的日期

    SELECT LAST_DAY('str');
> NULL

十八、date_sub(date,INTERVAL expr1 type)

date_sub(date,INTERVAL expr1 type) : 从日期里面减去指定的时间间隔。date参数是日期表达式，expr1参数是你希望减去多少天。

Type取值

例：

  • 希望2018-12-29 16:25:46.635 减去两天

    SELECT DATE_SUB('2018-12-29 16:25:46.635',INTERVAL 2 DAY);
> 2018-12-27 16:25:46.635

十九、date_add(date,INTERVAL expr type)

就是加几天

二十、format(X,D)

format(X,D) : 对字段的显示进行格式化

例：

  • 格式化两位小数点

    SELECT FORMAT(10000,2);
> 10,000.00

  • 格式两位小数点后会进行四舍五入

    SELECT FORMAT(42376482.234623,2);
> 42,376,482.23

二十一、date_format(date,format)

date_format(date,format) : 用于以不同的格式显示日期数据。format参数是输出的日期时间格式。date参数是日期类型

例：

  • 查看今天并精确到天

    SELECT DATE_FORMAT(NOW(),'%m-%d-%Y');
> 07-03-2022

  • 查看今天，精确到分和时段

    SELECT DATE_FORMAT(NOW(),'%b %d %Y %h:%i:%p');
> Jul 03 2022 03:15:PM

二十二、get_format(DATE|TIME|DATETIME,’EUR’|’USA’|’JIS’|’ISO’|’INTERNAL’)

get_format(DATE|TIME|DATETIME,’EUR’|’USA’|’JIS’|’ISO’|’INTERNAL’) : 返回一个时间格式字符串。一般都结合date_format()使用

例：

  • 想取精确到天的格式

    SELECT GET_FORMAT(DATE,'USA');
> '%Y-%m-%d'
    • 这是结合date_format()使用
    SELECT DATE_FORMAT(NOW(),GET_FORMAT(DATE,'USA'));
> 2022-07-03

  • 想取精确到秒的格式

    SELECT GET_FORMAT(DATETIME,'USA');
> '%Y-%m-%d-%H.%i.%s'

二十三、timestampdiff(interval,datetime_expr1,datetime_expr2)

timestampdiff(interval,datetime_expr1,datetime_expr2) : 返回表达式datetime_expr1日期时间和datetime_expr2日期时间之间的整数差。

例：

  • （两个时间之间）差了几个月

    SELECT TIMESTAMPDIFF(MONTH,'2003-02-01','2003-05-01');
> 3

  • （两个时间之间）相差几年

    SELECT TIMESTAMPDIFF(YEAR,'2002-05-01','2001-01-01');
> -1 (注意参数顺序，值是负数了)

二十四、least(value1,value2,…)

least(value1,value2,…) ： 返回值为最小值的参数

例：

  • 想在参数中取最小的值

    SELECT LEAST(2,0);
> 0

  • 想在参数中取最小的字母

    SELECT LEAST('B','A','C');
> 'A'

二十五、greatest(value1,value2,…)

greatest(value1,value2,…)返回值为最大值参数

二十六、find_in_set(str,strlist)

find_in_set(str,strlist) : 查询str 在strlist的位置，strlist要以英文逗号 , 分割

例：

二十七、unix_timestamp([date])

unix_timestamp([date]) ： 将时间转化为时间戳

二十八、from_unixtime(unix_timestamp)

from_unixtime(unix_timestamp) : 格式化时间戳

二十九、coalesce(value,…)

coalesce(value,…) : 返回值为列表当中的第一个非NULL 值，如果值全为null则返回 null

三十、instr(str,substr)

instr(str,substr) : 返回substr在字符串str第一个出现位置，如果没有则返回 0

三十一、locate(substr,str,[pos])

locate(substr,str,[pos])： 返回substr在字符串str第一个出现位置，如果没有则返回0

三十二、round(X,[D])

round(X,[D]) ： 返回参数 X最近的整数。参数 D可选的，表示保留小数点后几位

三十三、substring(str,pos,[len])

substring(str, pos)：从字符串的第 pos 个字符位置开始取，直到结束。

substring(str,pos,len)： 从字符串的第 pos 个字符位置开始取，取 len 个字符。

例：

  • 从字符串的第 4 个字符位置开始取，直到结束。

    select substring('example.com', 4);
> mple.com

  • 从字符串的第 4 个字符位置开始取，只取 2 个字符。

    select substring('example.com', 4, 2);
> mp

  • 从字符串的第 4 个字符位置（倒数）开始取，直到结束。

    select substring('example.com', -4, 2);
> .c

注意：在函数 substring(str,pos, len)中， pos 可以是负值，但 len 不能取负值。

三十四、substring_index(str,delim,count)

substring_index(str,delim,count) : 返回字符串 str 中在第 count 个分隔符 delim 之前的子串

如果 count 是一个正数，返回从最后的（从左边开始计数）分隔符到左边所 有字符。 如果 count 是负数，返回从最后的（从右边开始计数）分隔符到右边所有字 符。

例：

  • count 是正数，返回从分隔符“.”到左边的所有字符

    SELECT SUBSTRING_INDEX('www.generaldata.com.cn', '.', 2) FROM t;
> www.generaldata

  • count 是负数，返回从分隔符“.”到右边的所有字符

    SELECT SUBSTRING_INDEX('www.generaldata.com.cn', '.', -3) FROM t;
> generaldata.com.cn
